Introdução a Linguagem de Marcação

Endereços das Páginas Web

"http://www.univesp.br/cursos"

- http:// é o protocolo utilizado, Hypertext Transfer Protocol

- www.univesp.br indica o nome do "servidor" e que o endereço está registrado no Brasil

- /cursos serve para separar o nome do servidor do nome e diretório da página Web

As páginas são construídas usando linguagens de marcação, a mais conhecida é o HTML
(HyperText Markup Language)

O HTML é um conjunto de inúmeros comandos que identifica os elementos em uma página, como texto, imagem
arquivo, etc. O HTML é basicamente o esqueleto de uma página, posicionando cada elemento e arquitetando a 
página. É o que permite o browser exibir corretamente a página.

<head> é onde inserimos as informações para ajudar o navegador entender informações da página como: caracteres, lingua do pais, locais onde os arquivos estão armazenados

<html> marca o inicio e fim de um documento(Página)

<body> é basicamente o corpo da página, onde tudo que será exibido deverá ser colocado

<title> Título da página

<a> delimita links, imagens. o 'href' é o link da página desejada

<p> paragrafo comum.
